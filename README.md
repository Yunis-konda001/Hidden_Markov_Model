# HMM Activity Recognition

I built a Hidden Markov Model to recognize human activities from phone sensor data.

## Activities

- standing
- walking
- jumping
- still

## Data

Sensor Logger CSV files are in `data/` (accelerometer and gyroscope, ~100 Hz).

## How to run

```bash
pip install -r requirements.txt
jupyter notebook hmm_activity.ipynb
```

Run all cells top to bottom. Figures save to `figures/`.

## Files

- `hmm_activity.ipynb` — main analysis notebook
- `data/` — labelled sensor recordings
- `report/formative2_hmm_report.md` — report source (export to PDF for submission)
- `figures/` — plots generated by the notebook
